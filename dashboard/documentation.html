Dashboard Documentation

This dashboard provides a comprehensive interface for evaluating feature selection algorithms. It visualizes performance trends, statistical rankings, computational scalability, and selection stability.

Main Controls

  • Dataset: Filters all charts to a specific target dataset.
  • Metric: CLSACC, NMI, ACC, AUC, or AAD.
  • Range: Toggle between 10Percent (high resolution) and 100Percent (full spectrum) scales.

Core Features

Performance Curves Visualizes accuracy or error metrics as a function of the feature selection ratio.
Stability Analysis Displays a bar chart representing the Consistency Index of selected features across data perturbations.
Statistical Ranking Critical Difference (CD) Diagrams based on average ranks across all datasets to confirm statistical significance.
Computational Scalability Runtime analysis with linear and logarithmic scaling to identify algorithm bottlenecks.

Data Naming Conventions

The dashboard relies on strict naming to parse files automatically:

Performance Data

{Method}_{Metric}_{Range}.csv

Example: VCSDFS_AAD_10Percent.csv

Stability Data

{Method}_Stability_{Metric}_{Range}.csv

Example: VCSDFS_Stability_ACC_100Percent.csv

Columns Required: Performance files must have a Dataset column and numeric percentage columns (e.g., '0.05'). Stability files must have Dataset and Stability columns.

Detailed Score Table

The score table provides the raw data behind the curves. It includes an average score column which represents the area under the performance curve (based on FSDEM), used for overall ranking.

Publication Tools

Charts can be exported directly to PDF for inclusion in research papers. The dashboard also automatically styles the "Best" result in the data table for easy identification during analysis.
