Benchmarking with FSEVAL

The sdufseval library allows you to reproducibly benchmark feature selection methods on high-dimensional datasets, analyze their runtime scalability, and compute selection stability.

Installation

Install the package directly from PyPI:

Install via pip
pip install sdufseval

Full Benchmark, Stability, and Runtime Example

This script executes a complete evaluation pipeline:

  • Performance evaluation (CLSACC, NMI, ACC, AUC, and AAD)
  • Stability of feature selection algorithms based on each performance metric
  • Runtime scalability experiments
benchmark.py
from sdufseval import FSEVAL import numpy as np if __name__ == "__main__": DATASETS_TO_RUN = ['ALLAML', 'COIL20', 'Prostate_GE', 'colon', 'lung'] # Initialize FSEVAL evaluator = FSEVAL( output_dir="results", avg_steps=10 ) methods_list = [ { 'name': 'Random', 'type': 'unsupervised', 'func': evaluator.random_baseline }, { 'name': 'VCSDFS', 'type': 'unsupervised', 'func': lambda X: my_vcsdfs_logic(X) } ] # ─── 1. Run Standard Benchmark (Calculates ACC, AUC, AAD, etc.) ─── evaluator.run(DATASETS_TO_RUN, methods_list) # ─── 2. Run Scalability Analysis ─── print("\n>>> Starting Scalability Analysis...") evaluator.timer(methods=methods_list, vary_param='both')

Output

After running the script, the following files are generated for the dashboard:

  • results/ folder containing performance metrics (including Method_AAD_Range.csv).
  • Stability result files: Method_Stability_Metric_Range.csv (e.g., VCSDFS_Stability_ACC_100Percent.csv).
  • Runtime scalability CSV files (time_analysis_features.csv and time_analysis_instances.csv).
